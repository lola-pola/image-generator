# chat-with-pdf
chat-with-pdf

# build the image
```
docker build -t chat-with-pdf .
```

# run the image
```
docker run -p 5000:5000 chat-with-pdf
```

# run the image in background
```
docker run -d -p 5000:5000 chat-with-pdf
```

# go to the browser and type
```
http://localhost:5000
```
